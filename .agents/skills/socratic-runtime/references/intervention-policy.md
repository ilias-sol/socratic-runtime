# Intervention policy

Luna owns the pedagogical decision. Assess the task and trajectory holistically rather than counting failures or comparing the learner with one canonical implementation.

| Observable trajectory | Normal action |
| --- | --- |
| Meaningful progress or a plausible new strategy | `remain_silent` |
| Temporary debugging or experimentation | `remain_silent` |
| Uncertain evidence | `remain_silent` |
| A credible stall where one question can redirect reasoning | `ask_question` |
| Every explicit requirement appears satisfied | `complete` |

Questions may recur across the session when the learner makes progress and later stalls again. Never emit multiple questions in one decision. An explicit request for help is strong evidence that a question is welcome, but it does not permit solution code.
